# The Unfinished Promise

AS1 U.S. History digital museum exhibit.

Live site: https://edwardjunelee.github.io/

The exhibit presents six chronological units:

1. The 13 Colonies
2. Colonial Times / A New Era
3. Westward Expansion
4. Civil War
5. Gilded Age
6. Progressive Era
7. Conclusion

Each room is published as its own page and is accessible from the navigation tabs.
